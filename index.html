The id attribute should be your server's id or any server with widgets enabled.

Example of excluded id

All html attributes besides id are completely optional

Attribute Description
id Server ID *required
width Width of widget *optional; default = 350px
height Height of widget *optional; default = 500px
footerText Footer text *optional
color Join button and header color *optional; default = #5865f2
backgroundColor Main background color *optional; default = #0c0c0d
textColor Main text color *optional; default = #ffffff
statusColor User status text color *optional; default = #858585
